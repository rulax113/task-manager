# Aplikacja webowa REST API do zarządzania zadaniami (Task Manager).Task Manager

Prosta aplikacja webowa CRUD do zarządzania zadaniami.

## Technologie

* Node.js
* Express
* Git
* Docker
* Jest

## Uruchomienie lokalne

npm install
npm start

## Testy

npm test

## Architektura

Aplikacja oparta o REST API z pamięcią lokalną.

CRUD functionality tested using Thunder Client.

Automated tests were executed using Jest and Supertest.
